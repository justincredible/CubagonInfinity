# Cubagon Infinity

This is a minuscule [Stack](https://docs.haskellstack.org/en/stable/) project to demonstrate using OpenGL with Haskell.

#Linux
Some libraries will need to be installed:
GL, GLU, Xi, Xrandr, Xxf86vm, Xcursor, Xinerama

`stack run`
